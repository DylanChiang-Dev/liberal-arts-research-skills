# eval：cite-format

## 基準輸入

見 [examples/2026-06-14-citeformat-thesis.md](../examples/2026-06-14-citeformat-thesis.md)：作者碩論參考文獻 47 筆，指定整理為 APA 第 7 版。

## ✅ 必須做到（MUST）

- 明確提醒先用 `citation-verify` 查核真偽，再做格式整理。
- 只管格式，不驗證文獻是否真實，也不判斷引文是否支持論點。
- 缺頁碼、缺編者、缺地點、缺日期等欄位一律標註請補。
- 重複條目用標題等資訊判定，不只看作者＋年份。
- 若有正文，必須檢查隨文引註與文末清單是否一一對應。

## ⛔ 必須不做（MUST NOT）

- 不得為了湊格式編造頁碼、地點、作者、年份或 DOI。
- 不得把未查核或錯誤資料排漂亮後當作已正確。

## 已暴露的坑（防重犯）

- 先排後驗會把錯作者、錯年份排成漂亮格式；格式不能讓錯資料變真。
- API 覆蓋外類型是 cite-format 最容易出錯、也最需要人工補欄位的主場。
