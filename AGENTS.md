# AGENTS

本倉庫是《文組生的 AI 研究工作流》隨書開源技能庫，與書稿同步迭代。

- 技能規範：單文件 `skills/<name>/SKILL.md`，繁體中文優先，frontmatter 含 name 與 description。
- 內容鐵律：絕不編造（查核類技能尤其），所有工作流原創撰寫，思路來源在 README 致謝具名。
- 書稿項目（私有知識庫）：`/Volumes/001/01.projects/2026/20260612-第一本書-AI研究工作流--active/`，技能與書章對應表見 README。
- 版本策略：0.0.X＝打磨輪（實測修訂一輪尾號+1）；0.X.0＝新技能發布（對應一章書稿完成）；1.0.0＝書出版定版。每版打 git tag，版本史記於 MEMORY.md。
- 版權模式：倉庫內全部內容 MIT（版權人 Dylan Chiang／蔣濤，外部貢獻按 inbound=outbound 同樣以 MIT 授入）；書稿正文為傳統版權保留，與本倉庫互不影響；書名與品牌不在 MIT 授權範圍內。
